Texto da pergunta

O número decimal 32768 pode ser representado como (escolha todas as opções aplicáveis):

 

Pergunta 1Selecione um ou mais:
UM.

Palavra completa 00008000

B.

Palavra completa FFFF8000

C.

Meia palavra assinada 8000

E.

Palavra dupla 00000000FFFF8000

E.

Palavra dupla 0000000000008000

Opinião

Decimal 32768 para hexadecimal:


32768 / 16 = 2048, com resto 0 

  2048 / 16 = 128, com resto 0 

    128 / 16 = 8, com resto 0 

         8 / 16 = 0, com resto 8 


Decimal 32768 é hexadecimal 8000 .


Isso pode ser representado como palavra completa 00008000 ou palavra dupla 0000000000008000 .


Observe que o decimal 32768  não pode ser representado como meia palavra assinada 8000  (binário 1 000 0000 0000 0000) porque esse é um número negativo: -32768 . 


As respostas corretas são: Palavra completa 00008000, Palavra dupla 0000000000008000

Texto da pergunta

O número decimal -32768 pode ser representado como (escolha todas as opções aplicáveis):

 

Pergunta 2Selecione um ou mais:
UM.

Palavra dupla FFFFFFFFFFFF8000

B.

Palavra dupla 0000000000008000

C.

Palavra completa FFFF8000

E.

Palavra completa 00008000

E.

Meia palavra 8000

Opinião

-32768 é, em uma halfword binária, 8000 . O bit 0, o bit de sinal, é 1 (hexadecimal 8 é binário 1 000), indicando um número negativo.

Para representar o número como uma fullword , propagamos o sinal ( binário 1 ) para a esquerda:  FFFF8000 .

O mesmo para representar o número como uma doubleword : FFFFFFFF FFFF8000 .

As respostas corretas são: Meia palavra 8000, Palavra dupla FFFFFFFFFFFF8000, Palavra completa FFFF8000

O número decimal 4096, convertido para hexadecimal e colocado em uma meia palavra, é:

Opinião

4096 / 16 = 256 com resto 0

Pergunta 4

Incorreto
0,00 pontos de 1,00
Pergunta de bandeira

Texto da pergunta

A meia palavra assinada A000 pode ser representada em uma palavra completa como 0000A000

 

Pergunta 4Selecione uma:

Opinião

Para converter uma meia palavra em uma palavra inteira, propagamos o sinal para a esquerda.


Na meia palavra A000, bit 0, o sinal, é '1', o que significa que A000 é negativo (hexadecimal 'A' é binário ' 1 010'). Devemos propagar ' 1 ' para a esquerda.

Portanto, a representação correta de 'A000' como uma palavra completa é 'FFFFA000' ('F' é binário ' 1111 ').

Tanto A000 quanto FFFFA000 são decimais -24.576 .


A palavra completa 0000A000 é o decimal 40.960 , portanto não é o mesmo que a meia palavra A000.


A resposta correta é "Falso".

Pergunta 5

Incorreto
0,00 pontos de 1,00
Pergunta de bandeira

Texto da pergunta

O número decimal -8.000 pode ser representado como (escolha as duas respostas corretas)

 

Pergunta 5Selecione um ou mais:
UM.

Palavra completa FFFFE0C0

B.

Meia palavra 1F40

C.

Palavra completa 00001F40

E.

Palavra dupla FFFFFFFFFFFFE0C0

E.

Palavra dupla 00000000FFFFE0C0

Opinião

Para encontrar a representação hexadecimal de -8000, primeiro convertemos 8000 para hexadecimal e depois alteramos seu sinal (encontramos seu complemento de dois):


Decimal 8000 para hexadecimal:


8000 / 16 = 500, com resto 0
   500 / 16 = 31, com resto 4
     31 / 16 = 1, com resto 15 (hex F ) 
      1 / 16 = 0, com resto 1


Decimal 8000 é hexadecimal 1F40 . Convertendo 1F40 para negativo:


Positivo: 1F40 0001 1111 0100 0000

Inverter bits: E0BF 1110 0000 1011 1111 

Adicione 1:            E0C0       1 110 0000 1100 0000 (observe o sinal)                      


Para representar uma palavra completa , propagamos o sinal: FFFFE0C0

O mesmo para uma palavra dupla : FFFFFFFFFFFFE0C0

As respostas corretas são: Palavra dupla FFFFFFFFFFFFE0C0, Palavra completa FFFFE0C0
